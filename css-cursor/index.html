How to use CSS cursors

Set the cursor on any element with the cursor property: cursor: pointer;. It accepts a single keyword or a comma-separated fallback list ending with a keyword.

You can also use a custom image: cursor: url('my.cur') 4 4, pointer; — the numbers are the hotspot coordinates (x y), and the keyword at the end is the mandatory fallback if the image fails to load.

The none value hides the cursor entirely — useful for custom game UIs or full-screen immersive experiences. Pair it with JS to render your own cursor element for the best results.
