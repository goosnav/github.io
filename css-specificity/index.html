How specificity works

A — ID selectors
#id counts as (1,0,0). Higher A always wins.

B — Classes, attributes, pseudo-classes
.class, [attr], :hover, :nth-child() → (0,1,0) each.

C — Elements & pseudo-elements
div, p, ::before, ::after → (0,0,1) each.

* and combinators
Universal selector (*) and combinators (+, ~, >, space) add zero specificity.
