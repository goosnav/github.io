Quick Reference

list-style-type

Controls the bullet or number marker on <li> elements. Works on both <ul> and <ol>.

counter-reset

Creates a new counter (or resets it to a value) on the parent element.

counter-increment

Advances the counter by N on each matched element. Negative values count down.

counter() vs counters()

counter(name) reads the nearest counter. counters(name,".") concatenates all ancestor levels — useful for "1.2.3" outlines.

::before content

Inject the counter value as a pseudo-element. Combine with strings like "Step " counter(n) ": ".

list-style: none

Remove all markers. Pair with custom counters via ::before for full design control without browser defaults.
