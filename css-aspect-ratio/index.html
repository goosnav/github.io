How It Works

    

      

        
Modern: aspect-ratio

        
Set aspect-ratio: 16 / 9 on any element. The browser calculates the height automatically from the element's width. No wrapper needed. Supported in all modern browsers since 2021.

      

      

        
Legacy: padding-bottom hack

        
Percentage padding is relative to the width of the element, not the height. So padding-bottom: 56.25% on a zero-height wrapper always maintains a 16:9 ratio. Children are positioned absolute inside the wrapper.

      

      

        
Padding formula

        
padding % = (height ÷ width) × 100. For 16:9: (9 ÷ 16) × 100 = 56.25%. For 4:3: (3 ÷ 4) × 100 = 75%.
