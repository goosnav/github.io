How @media queries work

min-width

Applies when viewport is at least this wide. Mobile-first approach: start with base styles, add min-width overrides for larger screens.

max-width

Applies when viewport is no wider than this. Desktop-first approach: start large, add max-width overrides for smaller screens.

AND vs OR

and — all conditions must be true. , (comma) acts as OR — any condition can match. Mix them for precise targeting.

orientation

landscape — width > height. portrait — height ≥ width. Useful for mobile layout shifts.

prefers-color-scheme

Detects OS-level dark/light preference. Use dark to style for users who prefer dark interfaces.

prefers-reduced-motion

Detects users who prefer less animation. Always wrap decorative animations in reduce checks for accessibility.
